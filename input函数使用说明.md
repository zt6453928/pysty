# 🎯 input() 函数使用说明

## ✅ 好消息！

你的代码编辑器**已经支持 input() 函数**！

---

## 🎮 如何使用

### 步骤1: 点击"标准输入"按钮

在代码编辑器右上角，点击**"标准输入"**按钮。

### 步骤2: 输入数据

会出现一个文本框，**每行一个输入值**，对应每个 `input()` 调用。

### 步骤3: 运行代码

点击**"运行代码"**按钮，代码会使用你提供的输入数据。

---

## 📝 示例

### 示例1: 单个输入

**代码**:
```python
name = input("请输入姓名: ")
print(f"你好, {name}!")
```

**标准输入框**（点击"标准输入"按钮后）:
```
张三
```

**输出**:
```
你好, 张三!
```

---

### 示例2: 多个输入

**代码**:
```python
first_name = input("请输入名字: ")
last_name = input("请输入姓氏: ")
country = input("请输入国家: ")
age = input("请输入年龄: ")

print(f"姓名: {first_name} {last_name}")
print(f"国家: {country}")
print(f"年龄: {age}")
```

**标准输入框**（每行一个值）:
```
张三
李四
中国
25
```

**输出**:
```
姓名: 张三 李四
国家: 中国
年龄: 25
```

---

### 示例3: 数字输入

**代码**:
```python
a = int(input("请输入第一个数: "))
b = int(input("请输入第二个数: "))
print(f"{a} + {b} = {a + b}")
```

**标准输入框**:
```
10
20
```

**输出**:
```
10 + 20 = 30
```

---

## 🔍 重要提示

### 输入顺序必须匹配

**代码中的 input() 顺序**:
```python
first_name = input()  # 第1个
last_name = input()   # 第2个  
age = input()         # 第3个
```

**标准输入框中的顺序**:
```
张三    ← 第1个input()使用
李四    ← 第2个input()使用
25      ← 第3个input()使用
```

### 输入数量必须匹配

**代码有4个 input()**:
```python
a = input()
b = input()
c = input()
d = input()
```

**标准输入必须有4行**:
```
值1
值2
值3
值4
```

**如果输入少了**:
```
值1
值2
```
❌ 会导致 EOFError 错误（没有足够的输入）

## 💡 最佳实践

### 推荐方式

**开发时使用标准输入**:
```python
# 代码
name = input("姓名: ")
age = input("年龄: ")

# 标准输入框
张三
25
```

**提示清晰**:
- ✅ 知道要输入什么
- ✅ 输入顺序明确
- ✅ 易于测试

### 调试技巧

**如果遇到 EOFError**:

1. 检查代码有几个 `input()`
2. 数一数标准输入框有几行
3. 确保数量一致
4. 每行一个值，不要空行

## 🐛 常见错误

### 错误1: EOFError

**错误信息**:
```
EOFError: EOF when reading a line
```

**原因**: 输入数据不够

**解决**:
```
检查：
- 代码有几个 input()？
- 标准输入框有几行？
- 是否一致？

修复：
添加足够的输入行
```

### 错误2: 类型错误

**代码**:
```python
age = int(input())
```

**标准输入**:
```
二十五  ← 错误！应该是数字
```

**解决**:
```
25  ← 正确！
```
