# 🏆 成就系统 - 快速启动

## 一键初始化

```bash
# 1. 设置数据库环境变量
export DATABASE_URL="your-neon-database-url"

# 2. 初始化成就系统
npm run init-achievements
```

## 🎮 如何使用

### 访问成就页面

启动应用后，点击导航栏的 **"成就"** 按钮：

```bash
npm run dev
# 访问 http://localhost:3000/achievements
```

### 解锁成就

成就会在你学习过程中自动解锁：

- ✅ **完成练习题** → 自动检查练习题成就
- ✅ **完成关卡** → 自动检查进度成就
- ✅ **连续学习** → 自动更新连续学习统计
- ✅ **特殊条件** → 自动触发特殊成就

### 查看统计

成就页面显示：
- 🏆 成就完成进度
- 🔥 连续学习天数
- 🎯 完成关卡数量
- ✨ 完成练习数量

- 🎨 稀有度渐变背景

## 🔧 数据库表

成就系统创建3个表：

1. **achievements** - 存储所有成就定义
2. **user_achievements** - 记录用户解锁的成就
3. **user_stats** - 追踪用户统计（连续学习等）

## 💡 提示

- 成就解锁会自动增加魔法点数
- 某些成就有隐藏触发条件
- 连续学习是解锁高级成就的关键
- 完成所有成就可获得超过3000点奖励
