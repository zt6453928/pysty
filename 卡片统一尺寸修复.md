
## 🎨 内容适应

### 标题处理

```typescript
className="text-xl font-bold mb-2 spell-text line-clamp-1"
```

- **尺寸**: text-xl（较小，适应固定高度）
- **行数**: 1行（line-clamp-1）
- **超出**: 显示省略号

### 描述处理

```typescript
className="text-purple-200 mb-4 text-sm line-clamp-2 flex-1"
```

- **尺寸**: text-sm（小号字体）
- **行数**: 最多2行（line-clamp-2）
- **超出**: 显示省略号
- **flex-1**: 占据剩余空间

### 魔法点数

```typescript
className="mt-auto"
```

- **位置**: 始终在底部
- **对齐**: 所有卡片的点数在同一水平线

## 🔧 技术细节

### CSS 属性

| 属性 | 值 | 作用 |
|------|-----|------|
| width | 100% | 填充grid列 |
| height | 280px | 固定高度 |
| display | flex | 弹性布局 |
| flex-direction | column | 垂直排列 |
| gap (grid) | 1.5rem | 卡片间距 |

### Flexbox 布局

```css
.card {
  display: flex;
  flex-direction: column;
  height: 280px;
}

.content {
  flex: 1;           /* 描述占据剩余空间 */
}

.points {
  margin-top: auto;  /* 固定底部 */
}
```
