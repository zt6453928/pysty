# 📝 交互式输入（input）功能支持说明

## 🎉 新功能

代码编辑器现在支持 Python 的 `input()` 函数，**像真正的 IDE 一样运行时弹窗输入**！

## 💡 工作原理

- 使用 **Pyodide**（WebAssembly Python）在浏览器中直接运行 Python 代码
- 当代码执行到 `input()` 时，自动弹出输入对话框
- 完全模拟真实的交互式 Python 环境

## 🚀 使用方法

### 1. 编写包含 `input()` 的代码

```python
# 获取用户输入
first_name = input("请输入名字: ")
last_name = input("请输入姓氏: ")
country = input("请输入国家: ")
age = input("请输入年龄: ")

print(f"姓名: {first_name} {last_name}")
print(f"国家: {country}")
print(f"年龄: {age}")
```

### 2. 点击"运行代码"按钮

等待 Python 运行环境就绪（右上角显示 ✓ 就绪）

### 3. 在弹出对话框中输入数据

- 代码执行到第一个 `input("请输入名字: ")` 时，会弹出对话框显示提示 "请输入名字:"
- 输入 "张三" 后点击确定
- 继续执行到第二个 `input("请输入姓氏: ")` 时，再次弹出对话框
- 输入 "李" 后点击确定
- 依此类推，直到所有输入完成

### 4. 查看输出结果

所有输入完成后，代码继续执行并显示结果：
```
姓名: 张三 李
国家: 中国
年龄: 25
```

## ✨ 特点

✅ **真实交互体验**：就像在本地 Python 环境中运行一样  
✅ **运行时输入**：不需要预先准备输入数据  
✅ **提示信息清晰**：弹窗会显示 `input()` 中的提示文字  
✅ **支持取消**：点击取消按钮会停止代码执行  
✅ **完全客户端执行**：代码在浏览器中运行，速度快且安全

4. 点击确定继续

## 🔧 技术架构

### 前端
- **Pyodide v0.24.1**：WebAssembly Python 解释器
- **自定义 input() 函数**：使用 `window.prompt()` 实现交互式输入
- **实时输出捕获**：捕获 `print()` 的所有输出

### 执行流程
```
用户点击运行
    ↓
加载 Pyodide（首次需要几秒）
    ↓
执行 Python 代码
    ↓
遇到 input() → 弹窗
    ↓
用户输入 → 继续执行
    ↓
显示输出结果
```

## ⚠️ 注意事项

1. **首次加载**：第一次使用时 Pyodide 需要几秒钟加载，请等待右上角显示 "✓ 就绪"
2. **必须输入**：如果点击取消，代码会抛出异常停止执行
3. **浏览器环境**：代码在浏览器中运行，某些需要系统权限的功能可能不可用
